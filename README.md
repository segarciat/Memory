# Memory
The card game 'Memory', created in pygame.

## How to play

Click the face-down cards. If you guess all pairs, you win! Press `p` to pause the game.



## Installation

The game requires Python 3.8. To set up the game

```
pip install -r requirements.txt
```

## Usage

```
py -3 main.py
```
